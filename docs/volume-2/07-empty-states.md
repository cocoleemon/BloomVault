# 🌸 Empty States

> *"An empty screen isn't the absence of content—it's the beginning of a journey."*

---

# Experience Goal

Transform moments without content into moments of encouragement.

Every empty state should reassure users, communicate purpose, and gently guide them toward the next meaningful action.

BloomVault should never make an empty library feel like a failure.

Instead, it should feel like possibility.

---

# Introduction

Empty states are the screens users see when there is little or no content to display.

Rather than treating these moments as placeholders, BloomVault treats them as opportunities to educate, encourage, and inspire.

Every empty state should feel warm, intentional, and aligned with BloomVault's identity as a personal beauty library.

---

# Empty State Principles

Every empty state should:

* Explain what users are seeing.
* Encourage the next meaningful action.
* Feel calm and welcoming.
* Reinforce BloomVault's personality.
* Avoid making users feel they've done something wrong.

An empty screen should always have purpose.

# Writing Guidelines

Every empty-state message should:

* Sound encouraging.
* Use simple language.
* Avoid technical jargon.
* Focus on possibilities rather than limitations.
* Clearly suggest the next action.

BloomVault should never make users feel lost or discouraged.

---

# Visual Guidelines

Empty states should feel light and inviting.

They may include:

* Minimal illustrations.
* Soft botanical elements.
* Delicate bookmarks.
* Floral accents.
* Library-inspired imagery.

Visuals should support the message without distracting from it.

---

# Voice & Tone

BloomVault speaks with warmth and quiet confidence.

The tone should be:

* Gentle
* Encouraging
* Helpful
* Calm
* Hopeful

The product should feel like a trusted companion rather than a system delivering notifications.

---

# Closing

Every library begins with an empty shelf.

BloomVault embraces these moments as opportunities rather than shortcomings.

By transforming empty spaces into invitations to explore, learn, and grow, the product creates an experience that feels optimistic from the very first interaction.

Every empty state should remind users that their beauty library is just beginning.
